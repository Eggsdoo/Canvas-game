Game Tutorial

Welcome to the best game you've ever played, Bullseye! If you want to be the best, read the instructions below:

  1. Use your mouse's left click button to move the bullseye (player).
  2. Move against an egg by positioning yourself opposite of the direction you want to go.
  3. Collect 5 eggs to win the game.
  4. Avoiding enemies and obstacles will help you win faster!
  5. If you lose too many hatchlings, the game is over!
  6. Press 'R' to restart the game at any time. (Don't restart if you're not doing well, that's just sad)
  7. Keep your eye on the bull!
